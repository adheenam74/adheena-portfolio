# Fraud Risk Analysis (Excel Project)

This project is a simulated fraud analytics case study created using Microsoft Excel.  
It demonstrates how to identify suspicious financial transactions using fraud flags, risk scoring, and pivot-based analysis.

---

## Overview

The dataset contains simulated card transaction records.  
The objective is to detect unusual or potentially fraudulent behavior by:

- Creating fraud indicator flags  
- Building a weighted risk score  
- Applying conditional formatting to highlight risky transactions  
- Summarizing patterns using PivotTables and PivotCharts  
- Presenting insights in a clean overview sheet  

---

## Techniques Used

- Data cleaning  
- Fraud flag creation  
    - High Amount Flag  
    - Foreign Transaction Flag  
    - Card-Not-Present Flag  
    - Repeat IP Address Flag  
- Weighted risk scoring  
- Conditional formatting (High Risk / Medium Risk)  
- PivotTables (Country, IP, Customer)  
- PivotCharts  
- Dashboard summary page (Project_Overview)

---

## Key Insights

- Suspicious IP detected: **45.66.77.10** (highest risk score)  
- High-value transactions flagged from **203.88.9.1**  
- India had the highest total risk score among countries  
- Customers with repeated high-risk patterns identified
