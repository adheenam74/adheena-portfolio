# AML & KYC Analytics Projects

This repository contains simple, beginner-friendly projects related to Anti-Money Laundering (AML) and Know Your Customer (KYC) analytics.

---

## Project: AML Transaction Monitoring (Excel)

A small simulated AML case study that identifies:
- High-value transactions  
- High-risk country activity  
- Possible structuring/smurfing patterns  

The project includes basic Excel formulas, conditional formatting, PivotTables, and a simple dashboard.
